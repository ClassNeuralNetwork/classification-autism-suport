Este projeto visa promover a inclusão e acessibilidade de crianças com Transtorno do Espectro Autista (TEA), oferecendo um sistema baseado em inteligência artificial para auxiliar na triagem precoce. Utilizamos redes neurais artificiais para classificar traços de TEA a partir de questionários comportamentais, além de uma **camada interpretável** que gera perfis e recomenda estratégias educacionais personalizadas.

## Funcionalidades Principais

- **Classificação de Traços de TEA**: O sistema é capaz de analisar respostas de um questionário comportamental (questões de A1 a A10) e classificar a probabilidade de traços de TEA com alta acurácia (100%).

- **Camada Interpretável**: A partir das respostas individuais, o sistema gera um perfil comportamental detalhado, identificando dificuldades específicas como: interação social, comunicação gestual, comunicação social, imaginação, atenção conjunta, empatia social, linguagem atípica, comunicação não verbal e atenção atípica.

- **Recomendação de Estratégias Educacionais**: Com base no perfil gerado, o sistema oferece recomendações pedagógicas personalizadas baseadas em evidências, auxiliando educadores e cuidadores na intervenção.

- **Explicabilidade (XAI)**: Utilizando técnicas SHAP e LIME, o sistema torna suas decisões transparentes, permitindo que profissionais compreendam quais fatores mais influenciaram cada predição.
